# C-LINER
### Problematica

Generalmente los estudiantes tienen dificultad para aprender conceptos de la física mecánica, como lo es el MRU. Por ende, es necesiro una guia mas visual para el entendimiento de esta rama, y como interactúan y se relacionan las variables físicas dentro del fenómeno, en un escenario mas practico.

### Objetivos Generales

Crear una herramienta computacional interactiva y visual en lenguaje C, que simule problemas relacionados con el MRU, permitiendo la entrada de diversas variables del MRU. Esto con el fin de facilitar la enseñanza y comprensión del MRU en estudiantes de educación media.
