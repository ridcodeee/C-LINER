# C-LINER
# Problematica

Generalmente los estudiantes tienen dificultad para aprender conceptos de la física mecánica, como lo es el MRU. Por ende, es necesiro una guia mas visual para el entendimiento de esta rama, y como interactúan y se relacionan las variables físicas dentro del fenómeno, en un escenario mas practico.

# Objetivos

#### 1-. Objetivo General: Desarrollar un programa en C que calcule de manera precisa el tiempo y distancia de colisión de 2 vehiculos (MRU)
     
### Objetivos Especificos:
   -Implementar las ecuaciones de mru al codigo en forma de funciones 
   
   -Programar una interfaz que muestre los resultados obtenidos
